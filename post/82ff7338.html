数据结构

本篇概述什么是数据结构、数据结构与算法的关系以及相关概念术语。

一、笔记

     数据结构
     数据 指能输入到计算机中并被处理的符号的总称。
     数据元素 是数据的基本单位，性质相同的同一类数据元素称为 数据对象。
     数据类型 指数据对象集合和数据集合相关联的操作集。
     数据结构 通常包含数据逻辑结构、物理结构以及数据相关联的操作。逻辑结构：线性结构、树结构、图；物理存储：顺序、链接、散列、索引结构。

     算法分析
     算法 通常包含有穷性、确定性、可行性三个性质，以一组有限的指令集完成某个目标并产生输出。本书涉及的部分算法：穷举法、贪心法、分治法、回溯法。
     空间复杂度 $S(n)$: 算法程序在执行时占用存储单元的长度。
     时间复杂度 $S(n)$: 算法程序在执行时耗费时间的长度。
     算法分析时通常参考最坏情况复杂度、平均复杂度，平均复杂度难以度量，通常评估算法的好坏参考最坏情况复杂度。
     复杂度渐进表示法
     $T(N) = O(f(N))$ 表示存在常数 $C > 0, n_0 > 0$ 使得当 $n \ge n_0$ 时有 $T(N) \leq Cf(N)$
     $T(N) = \Omega (g(N))$ 表示存在常数 $C > 0, n_0 > 0$ 使得当 $n \ge n_0$ 时有 $T(N) \ge Cf(N)$
     $T(N) = \Theta (h(N))$ 表示同时有 $T(N)==O(h(N))、T(N) = \Theta(h(N))$

     算法分析 若 $T_1(N) = O(f(N))、T_2(N) = O(g(N))$，则有：
     $T_1(N) + T_2(N) = max(O(f(N)), O(g(N)))$
     $T_1(N) \cdot T_2(N) = O(f(N) \cdot g(N))$
     若 $T(N)$是关于 n 的 k 阶多项式，则 $T(N)=\Theta(n^k)$，表示复杂度只受高阶影响。

二、习题

     1. 最大子列和问题
     给定 N 个整数的序列 {$A_1, A_2, … ,A_N$}，求函数 $ \sum\limits_{k=i}^j A_k $ 最大值。

方法一：穷举法

int MaxSubSeqSum1(int List[], int N) {
    int ThisSum, MaxSum;
    for (int i = 0; i < N; i++) {        // i 为子列左端
        ThisSum = 0;
        for (int j = i; j < N; j++) {    // j 为子列右端
            ThisSum += List[j];
            if (ThisSum > MaxSum)
                MaxSum = ThisSum;
        }
    }
    return MaxSum;
}

方法二：分治法
int Max3(int A, int B, int C) {
  return (A > B) ? (A > C ? A : C) : (B > C ? B : C);
}

int DivideAndConquer(int List[], int left, int right) {
  int MaxLeftSum, MaxRightSum;              // 左右子问题解
  int MaxLeftBorderSum, MaxRightBorderSum;  // 跨跃分界线解

  int LeftBorderSum, RightBorderSum;
  int center, i;

  // 递归终止
  if (left == right) {
    if (List[Left] > 0){
      return List[Left];
    } else {
      return 0;
    }
  }

  // 分治
  center = (left + right) / 2;
  MaxLeftSum = DivideAndConquer(List, Left, center);
  MaxRightSum = DivideAndConquer(List, center+1, right)

  // 求跨界的最大子列和
  MaxLeftBorderSum = 0; LeftBorderSum = 0;
  for (int i = center; i >= left; i++) {
    LeftBorderSum += List[i];
    if (LeftBorderSum > MaxLeftBorderSum){
      MaxLeftBorderSum = LeftBorderSum;
    }
  }  // 左边扫描结束

  MaxRightBorderSum = 0; RightBorderSum = 0;
  for (int i = center + 1; i <= right; i++) {
    RightBorderSum += List[i];
    if (RightBorderSum > MaxRightBorderSum) {
      MaxRightBorderSum = RightBorderSum;
    }
  }  // 右边扫描结束

  return Max3(MaxLeftSum, MaxRightSum, MaxLeftBorderSum + MaxRightBorderSum);
}

int MaxSubSeqSum2(int List[], int N) {
  return DivideAndConquer(List, 0, N - 1);
}

方法三：在线处理
int MaxSubSeqSum3(int List[], int N) {
  int ThisSum = MaxSum = 0;
  for (int i = 0; i < N; i++) {
    ThisSum += List[i];
    if (ThisSum > MaxSum) {
      MaxSum = ThisSum;
    } else if (ThisSum < 0) {
      ThisSum = 0;
    }
  }
  return MaxSum;
}

     2. Maximum Subsequence Sum
     给定 K 个整数的序列 {$N_1, N_2, … ,N_K$}，求最大子序列和 {$N_i, N_i+1, …, N_j$}，子序列满足条件 $1 \leq i \leq j \leq K$，当输入序列全是负数时，输出最大序列和为 0，输出序列第一个数和最后一个数。

import java.util.Scanner;
public class Main {

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int k = scanner.nextInt();

        int[] list = new int[k];
        for (int i = 0; i < k; i++) {
            list[i] = scanner.nextInt();
        }
        MaxSubseqSum(list);
    }

    public static void MaxSubseqSum(int[] list) {
        int MaxSum = list[0], thisSum = 0;

        int fristIndex = 0, lastIndex = 0;
        for (int i = 0; i < list.length; i++) {
            thisSum += list[i];
            if (thisSum > MaxSum) {
                lastIndex = i;
                MaxSum = thisSum;
            } else if (thisSum < 0) {
                thisSum = 0;
            }
        }
        
        if (MaxSum < 0) {
            MaxSum = 0;
            fristIndex = 0;
            lastIndex = list.length - 1;
        } else {
            int temp = MaxSum;
            for (int j = lastIndex; j >= 0; j--) {
                temp -= list[j];
                if (temp == 0) {
                    fristIndex = j;
                }
            }
        }

        System.out.println(MaxSum + " " + list[fristIndex] + " " + list[lastIndex]);
    }
}



     3. 二分查找
     实现二分查找算法。

裁判测试程序样例，本题仅编写二分查找算法即可

#include <stdio.h>
#include <stdlib.h>

#define MAXSIZE 10
#define NotFound 0
typedef int ElementType;

typedef int Position;
typedef struct LNode *List;
struct LNode {
    ElementType Data[MAXSIZE];
    Position Last; /* 保存线性表中最后一个元素的位置 */
};

List ReadInput(); /* 裁判实现，细节不表。元素从下标1开始存储 */
Position BinarySearch( List L, ElementType X );

int main()
{
    List L;
    ElementType X;
    Position P;

    L = ReadInput();
    scanf("%d", &X);
    P = BinarySearch( L, X );
    printf("%d\n", P);

    return 0;
}

/* 你的代码将被嵌在这里 */


Position BinarySearch(List L, ElementType X) {
  int head = 1, end = L->Last;
  int m, flag = -1;
  while (head <= end) {
    m = (head + end)/2;
    if ( L->Data[m] == X) {
      flag = m;
      break;
    } else if (L->Data[m] > X) {
      end = m - 1;
    } else if (L->Data[m] < X) {
      head = m + 1;
    }
  }
  if (flag == -1) {
    return NotFound;
  } else {
    return flag;
  }
}

     分治法复杂度分析
     时间复杂度
       $T(N)=2T(N/2)+N$
       $T_1 = 1$
       $T_2 = 4 = 2 \cdot 2$
       $T_4 = 12 = 4 \cdot 3$
       可推导至一般形式 $T(2^k)=2^k \cdot (k+1)$
       若分割至 N，则 $T(N) = N \cdot (k+1)$，带入 $k=logN$
       $T(N)=NlogN+N=O(NlogN)$
     空间复杂度
       递归而产生的空间复杂度 = 每次递归的空间复杂度 * 递归深度，单次递归需要的空间是一个与 N 无关的常量，计为 $O(1)$, 分成 2 个、4 个、N 个，则需要分 logN 次，则 $S(n)=O(logN)$
       算法整体复杂度：n 个数据，求左边空间要 n/2， 右边要 n/2，跨越分界线需要扫描一次，则 $S(n)=2n=O(n)$
